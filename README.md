> An app for the Breath of the Wild cooking system built with React and Typescript. Add ingredients to see the stats of the resulting dish if they were cooked.

### ✨ [DEMO](http://codyrose.co/botw-cooking)

## Install

```sh
npm install
```

## Usage

```sh
npm start
```

## Run tests

```sh
npm test
```
